Apple Catcher

by: Advait Choksi

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game for a few reasons. The first is that players of all ages could play because the instructions were easy to understand. The second is that the game was not very easy to make but also not too complex. The last being that I wanted some animation in the game instead of a game like trivia or a card game.

2. What I Learned

When a user was playing my game, it was randomly getting over so I had to do a little bit of debugging in order to fix it. Some feedback was to make the game more challenging as the game goes on, and making it easier by increasing the number of lives that the user has.

3. Architecture of My Game

There is one runner class, that creates the window and draws the panel. There is another class that is called the GamePanel which draws everything that is to be displayed on the window. After this there are the objects in my game which are the catcher and the apples. These both interact in another class called the ObjectManager, where you check for collision, keep track of score, and change speed based on the score.

4. Technical Implementation

This is the technical implementation of the collision detection which I used in my game. It looks at each apple and sees if the x is within the catcher's width range. If that is true it will also check if the y is at the right spot where it is touching and if that is true it makes the apple inActive so it can disappear.

5. Most Challenging Part

In the beginning when I was making the game I ran into the issue of the collision not working and the apples not disappearing when they touched the basket. It took me a while to fix but I had to change the way I was checking the collision and then it started working.

6. Future Fixes and Updates

I could add more user prompts to maybe ask the user what fruit they wanted to play with out of a few choices. I could also add some bad apples that you are not supposed to catch and if you do catch them you lose a life.
